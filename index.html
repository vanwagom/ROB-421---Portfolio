Individual Artifact 1: Touch Command

For this artifact, I attempted to use Misty's touch sensors to drive it around objects. Using these controls, Misty was able to drive around an obstacle placed in front of her, Turn around and go back to where she started, or do a full loop around the obstacle

Individual Artifact 2: Yep, that's a...

Using Misty's object detection, I had the robot identify multiple objects. These objects were a book, a person, and a backpack.

Individual Artifact 3: Strafe

For this artifact, I wanted to be able to get the pupper to move side to side using hardware interface commands. In theory this method would give me more precision and control in how the robot makes any individual move. However in practice, the pupper was far less stable and movement in any given direction was merely a suggestion with a bit of luck.
